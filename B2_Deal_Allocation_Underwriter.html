(From Tree B1: Application Ingestion Process)

LOS triggers the Deal Allocation process.

LOS BRK-01 FR-07
Subject Property Province = Quebec
  1. LOS checks whether a Broker profile with the same Profile ID exists in the system.
  2. LOS simultaneously checks whether a Brokerage with the same Firm Code exists in the system.
LOS FR-07 UWR-01
EITHER Broker or Brokerage NOT found
  1. LOS sets Application Status → Onboarding Broker.
  2. Automatic deal allocation is skipped.
  3. If Broker or Brokerage are already known, they are auto-associated to loan on LOS. Otherwise association is left blank.
  4. Onboarding Team notified to complete broker onboarding.
  5. Onboarding Team contacts broker to begin onboarding process. AMF licence number captured and verified.
  6. When onboarding is complete, Broker is assigned to Quebec Sales Region and set to Active.
  7. Once Broker and/or Brokerage are created in the system, Onboarding Team manually associates them to the loan on the LOS.
  8. Onboarding Team moves application milestone to Received.
Onboarding Team Salesforce BRK-02 FR-08
BOTH Broker & Brokerage already exist in system
  1. Broker is associated to the application.
  2. Brokerage is associated to the application.
  3. LOS checks: is the Broker–Brokerage relationship status Active with an assigned sales region?
LOS FR-07 UWR-01
Broker Inactive OR no Sales Region
  1. LOS sets Application Status → Onboarding Broker.
  2. Automatic deal allocation is skipped.
  3. Onboarding Team reviews Broker Status and/or assigns Sales Region.
  4. Once relationship is Active and Sales Region is assigned, future deals from this Broker will be automatically allocated to Underwriter.
  5. Onboarding Team moves application milestone to Received.
Onboarding Team BRK-02 FR-07
Broker Status is Active AND Sales Region assigned
  1. LOS sets Application Status → Received.
  2. LOS selects Underwriting Team assigned to same Sales Region as Broker.
  3. Once Team is identified, LOS checks if any Underwriter on that team is available.
LOS FR-07
No Team Available OR No Underwriter available
  1. Skip automated allocation.
  2. Deal remains in All Applications queue.
  3. Deal remains in Received status.
LOS Manager FR-07 U01
Team identified AND Underwriter is available
  1. LOS auto-allocates deal to the next available Underwriter on the identified Team (round-robin).
  2. LOS sets Application Milestone → Allocated.
  3. All users now see the allocated deal via Team View for the assigned Underwriter's team
LOS FR-07 U02 U04



Skip auto-allocation paths converge

  1. Manager monitors All Applications view for any new & unassigned deals that are in either Onboarding Broker or Received status.
  2. When a new deal is identified, Manager is able to review and manually assign the application to an Underwriter on the LOS.
Manager LOS U01



Manual & Auto-allocation paths converge
  1. Underwriter sees the allocated deal via Team View for their team, or via the @Me filter.
  2. Quebec-specific: Underwriter confirms Application Language Preference (FR-01) — set at ingestion from Primary Borrower language. Underwriter may toggle or manually override if required.
  3. Quebec-specific: Deal is visible using the Quebec Region filter (FR-19).
  4. Underwriter opens deal and manually updates loan milestone to In Review, and begins application review.
Underwriter LOS FR-01 FR-19 UWR-01

→ Allocation complete. Proceed to Tree C1: Underwriter Review & Decisioning.

Underwriter Review UWR-01 UWR-02
Manual Allocation — All Amber Paths: In all cases where automated allocation is skipped, the deal surfaces in the All Applications queue. A Manager assigns the deal to a Quebec bilingual Underwriter. The Manager can use the Region (Quebec) and Language filters to identify unallocated Quebec deals (FR-19). Once assigned, the user sets the application status and milestone manually.

Key Assumptions (AS)
AS-03: All client and broker communication defaults to French unless an English preference is explicitly stated. Onboarding Team contacts unregistered Quebec brokers in French first.
AS-08: Quebec broker onboarding is identical to rest-of-Canada except: outreach in French first, AMF licence number captured in Salesforce, and broker assigned to a Quebec Sales Region.
Q-23 (Resolved): All Quebec deals enter the allocation queue for team review — including non-Tier-1 property deals and BFS clients. No special filtering or auto-decline applies at the allocation stage.
